# Campus Care System

A Java-based desktop application built as a second-semester OOP project, designed to help university students manage their academic life efficiently.

## Features
- **Student Login System** – Secure authentication for personalized access
- **Academic Planner** – Track assignments and exams with deadlines
- **Quiz Module** – Practice quizzes with result tracking
- **Flashcard System** – Study aid for quick revision
- **Dashboard** – Centralized view of all academic activities

## OOP Concepts Applied
- Inheritance & Polymorphism
- Interfaces (`Saveable`)
- Encapsulation across model classes
- GUI with Java Swing

## Tech Stack
- Language: Java
- UI: Java Swing (GUI)
- Storage: File I/O (`.txt` files)

## How to Run
1. Open the project in IntelliJ or VS Code (with Java extension)
2. Navigate to `src/main/Main.java`
3. Compile and run
